# Flutter KT3

Небольшое Flutter-приложение, демонстрирующее работу со статическими ресурсами.

## Функциональность

* Отображение 3 изображений из папки `assets/images/`
* Использование кастомного шрифта из `assets/fonts/`
* Применение темы с пользовательским шрифтом

## Структура проекта

* `assets/images/` — изображения
* `assets/fonts/` — шрифт
* `lib/main.dart` — основной код приложения
* `pubspec.yaml` — подключение ресурсов

## Запуск

```bash
flutter pub get
flutter run
```
