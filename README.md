# Flutter KT4. Сохранение состояния

Приложение-счётчик на Flutter. При нажатии на кнопку значение увеличивается, и сохраняется между запусками приложения.

## Используемые технологии

* Flutter
* SharedPreferences (локальное хранение данных)

## Запуск проекта

```bash
flutter pub get
flutter run
```

## Проверка

1. Нажать кнопку несколько раз
2. Закрыть приложение полностью
3. Запустить снова
4. Убедиться, что значение сохранилось
