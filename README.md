# Flutter Todo App (KT2)

Простое кроссплатформенное приложение Todo List, разработанное на Flutter.

## Возможности

- Добавление задач (title + text)
- Отображение списка задач
- Удаление задач (свайпом или кнопкой)
- Модальное окно для создания задачи
- Работа со state (StatefulWidget)

## Запуск проекта

- Установить зависимости:
```
flutter pub get
```

- Запуск приложения:
```
flutter run
```
